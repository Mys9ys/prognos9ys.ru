# Game Design v0.1 — prognos9ys + усадьба + биржа + ККИ

Актуально на: **2026-06-22**  
Статус: черновик для согласования. CS2 на бою — пауза.

---

## 1. Принципы

- **Прогнозы ЧМ** — ядро; ставки и баллы **не портим** бонусами усадьбы/крафта.
- **Усадьба** — долгая цель; **гос. делянка** доступна без сертификата.
- **Казна** — центр ликвидности: 🪙, 💎, ресурсы, лавка, биржа.
- **Карточный футбол** — отдельный веб-проект, **общий аккаунт**, своя валюта (TBD).
- **Рублии** — не на бирже; только **банк** (курс покупки/продажи). Донат → позже.

## 2. Сундук за level-up (3 блока)

При открытии — **три независимых ролла**.

### Блок 1 — валюта
| Награда | Примечание |
|---------|------------|
| 25 / 50 / 100 🪙 | |
| 5 / 10 💎 | |

Вероятности — финализировать отдельно (сумма 100%).

### Блок 2 — прогрессия
| Награда |
|---------|
| Банка XP 25 / 50 |
| Сертификат на усадьбу (1 сертификат = 1 участок) |
| Сертификат на слот профессии (универсальный) |
| Премиум 1 / 3 / 5 дней |

- Лишние сертификаты — **биржа**; казна подкидывает при дефиците.
- **Премиум** выпадает **неактивированным** → можно **торговать** на бирже; активация — отдельное действие игрока.

### Блок 3 — коллекция (prognos9ys + задел ККИ)
| Награда | Где живёт |
|---------|-----------|
| Карточка игрока | Запакованный пак → ККИ (позже) |
| Карточка действия на поле | Пак `field_action` |
| Карточка действия тренера | Пак `coach_action` |
| Карточка тренера | Пак `coach` (редче) |
| Расстановка | Пак `formation` |
| Шарф клуба / сборной | Коллекция prognos9ys (стена в усадьбе), не ККИ-механика |

**Распаковка паков** — **не делаем**, пока нет пула игроков/тренеров/действий ЧМ-26.

Планируемые типы паков:
- `player` — игроки ЧМ-26
- `coach` — тренеры
- `formation` + `field_action` + `coach_action` — смешанный или раздельные паки действий/расстановок

## 3. Лавка казны (сундуки ЧМ-26)

**Не биржа** — персональная лавка у каждого игрока, деньги в казну.

| Событие | Содержимое |
|---------|------------|
| С **40 тура** | 1 сундук ЧМ-26 за **50 🪙** + 1 за **5 💎** |
| **50 тур** и далее | снова по 1+1 каждого вида **если выкупил на предыдущем этапе** |
| Не выкупил оба | на следующем туре **ничего не добавляется** |

Каждые **10 матчей** в лавке: премиум **1 сутки** за **5 💎** (неактивированный, торгуемый).

Содержимое сундука — **таблица ЧМ-26** (отдельно от level-up / матчевого).

## 4. Казна и биржа

### Админ-панель «Казна»
Доступ: админы, супер-модераторы.

- Остатки 🪙 / 💎
- Склад ресурсов (гос. добыча)
- Управление лотами (сертификаты, премиум-пакеты)

### Гос. вклад поддержки
- **500 🪙** под **5% на 5 туров** (как обычный вклад по сроку)
- После 5 туров — **выплата процентов**
- Завершение вклада — **кнопкой** (как у обычного)

### Кнопки досрочного выхода (банк)
| Действие | Условие |
|----------|---------|
| Досрочное изъятие вклада | есть ликвидность в банке, **потеря процентов** |
| Досрочное погашение займа | выплата **всех** процентов |
| Изъятие гос. вклада поддержки | отдельная кнопка |

### Биржа (MVP)
- Всё из **инвентаря игрока** можно выставить
- Комиссия продавца: **20%** (без премиума) / **5%** (с премиумом)
- Динамика цен: лёгкое снижение при избытке, рост при дефиците (BDO-логика, детали TBD)
- Редкие ресурсы (янтарь, мрамор…) — **торгуемые**

### Премиум на бирже (каждые 10 туров)
Казна выставляет пакеты:
| Лоты | Срок | Стартовая цена в 🪙 |
|------|------|---------------------|
| `(игроков / 5)` | 3 суток | по курсу **12 💎** |
| `(игроков / 10)` | 5 суток | по курсу **20 💎** |

Дальше — правила биржи. Лоты **неактивированного** премиума.

## 5. Усадьба и крафт

### MVP релиз
1. **Гос. делянка** (без сертификата)
2. **2 профессии** на выбор (остальные — сертификат + уровень игрока)
3. **Биржа** + аукцион заказов на работу

### Профессии (10 стартовых)

**Добыча (5):** лес, камень, руда, песок (стекло), хлопок (ткань)

**Обработка (5):** столяр (доски), каменотес (блоки), плавильщик (слитки), стеклодув (листы), ткач (ткань)

Позже — профессии для **уникальных материалов** (янтарь, мрамор…).

### Слоты профессий
- Старт: **2 любые**
- **+1 за уровень игрока** (уровень = XP за прогнозы): ур.0→2, ур.1→3…
- Сертификат профессии = **+1 слот** (универсальный)
- Уровень профессии **не связан** с уровнем игрока

### Idle / итерации
- **1 итерация = 5 мин** (офлайн = онлайн)
- **30 мин** без премиума ≈ **6 итераций**, затем **стоп**; игрок перезапускает вручную
- С ростом уровня профессии — **незначительное** ускорение итерации
- **Премиум:** очередь циклов, автопродолжение; сырьё с прошлого цикла → в следующий

### Гос. делянка
- **0,5 🪙 за итерацию** (не за штуку ресурса)
- Двойной дроп на итерации — бонус без доп. платы
- Работа «для государства» → ресурс на **госсклад** → аукцион
- Редкий дроп: **уровень профессии × 0,1%** (ур.2 → 0,2%)

### Рынок труда
Владельцы точек (позже — личные делянки) выставляют **оплату за итерацию**; работники берут заказы (30 мин или премиум-очередь).

### Усадьба
- Сертификат = **1 пустой участок**; готовую усадьбу можно **продать отдельно**
- Бонусы усадьбы → **крафт/фарм** (не прогнозы)
- Старт дома: сервант с кубками, стена с шарфами; двор (поле, хлопок…) — по мере реализации
- Редкое + **экипировка** добытчика/крафтера (скорость, % редкого)

### Улицы
- **20 участков** (10+10)
- При **15 застроенных** → **новая улица**
- Игрок **выбирает** старую или новую
- Возможны полупустые старые улицы
- Кланы / улица-клана — **TBD**

### Личные делянки
- Покупка за 🪙 — **после MVP**
- Уровни делянки → больше одновременных рабочих (1 ур. → 2, 2 ур. → 4…)

## 6. Карточный футбол (отдельный проект)

### Связь с prognos9ys
- Один логин / `user_id`
- Своя **пара валют** (TBD)
- Паки из сундуков prognos9ys — в инвентаре, **распаковка позже**
- Звёздные игроки: в prognos9ys стихийно, на площадке ККИ — **больше возможностей**

### Стартовая колода (регистрация в ККИ)
| Тип | Кол-во |
|-----|--------|
| Игроки (ноунейм) | 26 (3 ВР / 8 ЗЩ / 9 ПЗ / 6 НП) |
| Тренер | 1 |
| Действия тренера | 3 |
| Расстановка | 1 (4-4-2) |
| Действия на поле | 18 (дубли OK) |

### Статы игроков
- Полевые: Pace, Shooting, Passing, Dribbling, Defending, Physicality (+ подстаты внутри, на карте — 6 групп)
- ВР: Diving, Handling, Kicking, Reflexes, Speed, Positioning
- Скрытые: Composure, Work Rate, Weak Foot, Skill Moves…
- Редкость ↔ общий рейтинг карты (рост при прокачке)

### Онбординг
- **3 тестовых чемпионата** × 10 команд, дом/выезд, матчи в любое время
- Новый чемпионат — **со следующего дня**
- После 3 чемпионатов: тестовые лиги дают **10% XP** (песочница)
- Далее: регистрация в выходные, игры пн–пт; кланы, сборные

### MVP
- **1 игрок + 9 ботов**; боты **разнообразные**, **не качаются**
- Авто-симуляция матча; вмешательство в **перерыве** (замены, тренер)

### Матч (кратко)
- 2 тайма × 5 этапов; шахматное чередование А-З
- 6 заготовленных комплектов (3 атака + 3 защита) на тайм из пула действий
- Тренер: 1 карта в 1-м тайме, 2 в перерыве, 3 во 2-м → слот этапа
- Энергия: 15% за матч (2%/этап), перерыв +5%; &lt;30% → −20% статов; пропуск +40%
- Замены: **5**, с очередью на травму
- Овертайм: +2 этапа на тайм; пенальти — 5 назначенных + random
- Разрешение этапов (гол/отбой/контратака) — **проработать при реализации**
